# Branding 자료

**SubQuery의 브랜드 관련된 모든 것들은 상표 등록이 되어 있으며 우리는 우리 브랜드를 매우 중요하게 생각합니다.**

만약 상표, 로고, 디자인 또는 기타 브랜드 기능을 사용하고자 한다면, 지침을 주의 깊게 따르거나 소셜 미디어를 통해 당사에 연락하시길 바랍니다.

의문점이 있다면, 물어보십시오!
